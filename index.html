Security Product

Phishguard Enterprise

AI-powered Gmail phishing detection app.

Phishguard Enterprise helps users detect phishing emails and malicious links by securely analyzing Gmail messages using read-only access.

Test Instructions:

1. Download the APK from: https://phishguardenterprise.xyz/phishguardenterprise.apk

2. Install the app on an Android device.

3. Open the PhishGuard Enterprise app.

4. Enter VirusTotal API key.

5. Click "Scan".

6. Sign in with Google and grant Gmail read permission.

7. The app will fetch recent emails and scan them for phishing threats.
